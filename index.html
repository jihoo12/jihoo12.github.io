MyLang is a minimalist toy compiler built for the LLVM ecosystem. My mascot Mylly and I are here to help you build precision-focused applications.

================================================================================

01. ARITHMETIC

Native 32/64-bit integer and double-precision floating point operations optimized via LLVM backends.

02. CAST_AUTO

Smart implicit casting. Promotion of INT to DOUBLE and truncation of DOUBLE to INT handled at the IR level.

03. ASM_INLINE

Support for GCC-style inline assembly strings. Direct register manipulation and variable mapping.

--------------------------------------------------------------------------------

DOCUMENTATION

VARIABLE_DECL

Syntax: let <Type> <Name> = <Expr>

Supports: i32, i64, double

ASM_BLOCK

Syntax: asm "code", ops...

Uses GCC-style operands (%0, %1) for LLVM translation.
