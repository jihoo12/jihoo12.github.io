Fast Arithmetic

Native speed for i32, i64, and double operations with LLVM optimization passes.

Implicit Casting

Automatic promotion of integers to doubles and truncated assignments for seamless development.

Inline Assembly

Directly inject GCC-style LLVM assembly strings with variable operand mapping.

Clean Syntax, No Fluff

example.ml
// Mixed-type arithmetic with implicit casting
let i32 a = 10
let double b = 3.14
let double res = a + b // a is auto-promoted to double
 
// Inline assembly support
let i32 val = 10
asm "mov $20, %0", val
 
return res
